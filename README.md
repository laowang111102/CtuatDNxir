# 城市天气可视化统计分析系统 python985

## 项目概述

城市天气可视化统计分析系统是一个利用Python、Flask和Vue技术栈，实现对城市天气情况全面、详细的收集，并通过可视化手段展示天气数据的软件系统。

## 技术栈

- 开发工具：PyCharm、VSCode
- 技术运用：API集成、ECharts、Flask、HTML、Vue
- 数据库：MYSQL
- 相关组件：phpEnv

## 功能模块

系统主要由以下七大模块组成：

1. **登录注册模块**
   - 用户注册
   - 用户登录
2. **实时天气查询模块**
   - 查询实时天气
   - 生活指数展示
   - 数据可视化
   - 场景交互
   - 查看24小时预报
   - 未来七天预报
3. **历史天气查询模块**
   - 查询历史天气
4. **预警信息推送模块**
   - 查看预警信息
5. **用户管理模块**
   - 修改用户名
   - 修改密码
6. **实时天气地图**
   - 直观查看全国温度、体感温度、湿度、能见度、风力等级情况
7. **可视化大屏**
   - 历史天气可视化

## 运行环境

- Windows

## 部署步骤

1. 将`weather_py`项目导入PyCharm。
2. 将`weather_vue`项目导入VSCode。
3. 安装npm，配置相应环境。
4. 在`weather_vue`文件内部打开命令控制器，输入`npm run dev`以打开Vue-Vite项目界面。
5. 在PyCharm中运行`app.py`文件。
